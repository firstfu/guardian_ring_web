# 圖片資源說明

## 📱 來電截圖

請將您提供的來電截圖命名為 `mom-call-screenshot.png` 並放在此資料夾中。

### 步驟：
1. 將您的來電截圖保存為 `mom-call-screenshot.png`
2. 放入此 `public/images/` 資料夾
3. 重新載入頁面即可看到真實的截圖

### 檔案路徑：
```
public/
  images/
    mom-call-screenshot.png  ← 您的來電截圖放在這裡
    README.md               ← 本說明檔案
```

### 注意事項：
- 建議圖片尺寸：280x580 像素（或等比例）
- 檔案格式：PNG 或 JPG
- 檔案大小：建議小於 1MB 以獲得最佳載入效果
